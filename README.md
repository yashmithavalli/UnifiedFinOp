# Unified FinOps Platform

A centralized financial intelligence system for public and private sectors.

## Getting Started

### Prerequisites

- **Node.js**: Ensure you have Node.js installed.
- **MongoDB**: This project requires a running MongoDB instance.

### Installation

1. Install the dependencies:
   ```bash
   npm install
   ```

2. Make sure MongoDB is running on your machine (default port: `27017`).

### Running the Website

1. Start the server:
   ```bash
   node server.js
   ```

2. Open your browser and navigate to:
   [http://localhost:3000](http://localhost:3000)

## Features

- **Dashboard**: Real-time spending visibility.
- **Policies**: Automated budget validation and policy enforcement.
- **Alerts**: Detect irregular spending patterns.
- **Dual Module**: Tailored experiences for both Public and Private sectors.
